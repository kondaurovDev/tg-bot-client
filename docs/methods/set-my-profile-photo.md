# setMyProfilePhoto

Changes the profile photo of the bot

[Telegram docs](https://core.telegram.org/bots/api#setmyprofilephoto)

## Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| photo | [`InputProfilePhoto`](../types/input-profile-photo.md) | Yes | The new profile photo to set |

## Return type

`boolean`
