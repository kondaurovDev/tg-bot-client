This object represents an inline keyboard that appears right next to the message it belongs to.

[Telegram docs](https://core.telegram.org/bots/api#inlinekeyboardmarkup)

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| inline_keyboard | [`InlineKeyboardButton[]`](/api/types/inline-keyboard-button[]/)[] | Yes | Array of button rows, each represented by an Array of InlineKeyboardButton objects |
