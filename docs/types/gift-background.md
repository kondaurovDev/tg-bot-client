# GiftBackground

This object describes the background of a gift.

[Telegram docs](https://core.telegram.org/bots/api#giftbackground)

## Fields

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| center_color | `number` | Yes | Center color of the background in RGB format |
| edge_color | `number` | Yes | Edge color of the background in RGB format |
| text_color | `number` | Yes | Text color of the background in RGB format |
